## What is Sik AI?

Sik AI simulates a LiDAR body scanner that tracks 11 body measurements over time with a stunning dark UI (#0A0A0A), emerald accent (#10B981), and 8 weeks of pre-seeded analytics.

**Bundle ID:** `com.sikai.bodyscanner`  
**Framework:** Expo SDK 54 / React Native 0.81  
**Navigation:** Expo Router — 5 tabs, 9 screens  

## Run Locally in 60 Seconds

```bash
git clone https://github.com/Sanjay-995/sik-ai.git
cd sik-ai
pnpm install
cd artifacts/sikai
npx expo start
```

Scan the QR code with the **Expo Go** app ([iOS](https://apps.apple.com/app/expo-go/id982107779)) on the same Wi-Fi network.
